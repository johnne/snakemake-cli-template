# Cookiecutter Snakemake CLI package

This is a [cookiecutter](https://github.com/cookiecutter/cookiecutter) template
for a [Snakemake](https://github.com/snakemake/snakemake/) workflow running as a
command line tool using [snk](https://snk.wytamma.com/).

## Quickstart

Install [uv](https://docs.astral.sh/uv/getting-started/installation/) then run:

```bash
uvx cookiecutter gh:johnne/snakemake-cli-template
```

You will be prompted for information such as your name, email, GitHub username
and project name and description. When you're done a new directory with the
project name you entered will be generated.
